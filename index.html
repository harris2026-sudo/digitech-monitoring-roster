⚠️ Deteksi Konflik Cuti Bersamaan

Sistem otomatis menandai konflik jika ketiga nama berikut tercatat cuti pada tanggal yang sama (lintas divisi): Harris, Sony Salubongga, Nyco — memastikan minimal 1 dari mereka tetap standby. Tandai status cuti lewat panel detail tanggal di masing-masing tab divisi.

☁️ Penyimpanan Otomatis

Aplikasi ini menyimpan seluruh data (tim, rotasi per divisi, override, tanda cuti) secara otomatis ke penyimpanan server (window.storage) setiap ada perubahan — tidak perlu export/import manual, dan data tetap ada walau tab browser ditutup atau dibuka ulang. Tombol di bawah hanya untuk backup opsional.

📂 Restore dari Backup

Pilih file JSON hasil backup sebelumnya untuk memuat ulang data — hasil restore ini juga akan langsung disimpan otomatis.
